Food Recipe API App

A basic frontend web application built using HTML, CSS, and JavaScript that fetches and displays food recipes using a public Recipe API.  
This project was created to understand API integration, asynchronous JavaScript.

Features

-  Search food recipes by name
-  View recipe details fetched from an external API
-  Dynamic data loading using JavaScript
-  Simple and clean user interface
-  Fully frontend-based project

Technologies Used

- HTML5 – Structure
- CSS3 – Styling
- JavaScript – Logic & API handling
- Public Recipe API – Data source
- Git & GitHub – Version control

How to Run

1. Download or clone this repository
2. Open `index.html` in any web browser
